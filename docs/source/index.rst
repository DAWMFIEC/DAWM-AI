Desarrollo de Aplicaciones Web y Móviles - SOFG1006
===================================

**Desarrollo de Aplicaciones Web y Móviles** (/DAWM/) es una asignatura del programa de la carrera de `Computación <https://www.fiec.espol.edu.ec/es/carreras-de-grado/computacion>`_, de la `Escuela Superior Politécnica del Litoral (ESPOL) <https://www.espol.edu.ec/es>`_, cuyo propósito es diseñar e implementar aplicaciones web y
móviles cumpliendo con los estándares actuales y las buenas prácticas de programación.
